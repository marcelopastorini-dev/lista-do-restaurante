# 🍕 Lista de Produtos em Falta - Restaurante & Pizzaria

Este é um projeto simples desenvolvido em HTML, CSS e JavaScript para gerenciar e visualizar rapidamente os insumos e ingredientes que estão em falta no estoque do restaurante/pizzaria.

## 🚀 Funcionalidades
* **Visualização Rápida:** Lista clara dos ingredientes esgotados para agilizar as compras.
* **Interface Dinâmica:** Manipulação dos itens em tempo real via JavaScript.
* **Foco Operacional:** Desenvolvido para ajudar a equipe de cozinha e atendimento.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação da página e dos elementos da lista.
* **CSS3:** Estilização da interface (responsiva para telas de celulares ou tablets).
* **JavaScript (ES6):** Lógica para adicionar, remover ou listar os produtos dinamicamente.


Desenvolvido para otimizar a rotina e o controle de estoque do restaurante.
